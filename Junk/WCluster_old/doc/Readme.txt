===============================================================================

　WCluster

===============================================================================


■ソフトの概要

　フォルダ・ファイルの圧縮・展開を行います。いわゆるアーカイバです。

　一度に圧縮できる対象は単一のフォルダ又は単一のファイルです。
　アーカイブファイルはこのアプリ固有の形式です。（拡張子は wclu です）
　暗号化に対応しています。

　暗号は共通鍵（秘密鍵）方式を採ります。
　鍵を生成し、安全な輸送手段で共有した上で、例えば・・・

　　●ＡとＢ両方の鍵を持っている人のみ復号可能
　　●Ｃ、Ｄ、Ｅ何れかの鍵を持っている人のみ復号可能
　　●「ＦとＧ両方の鍵を持っている」又は「Ｈの鍵を持っている」人のみ復号可能

　・・・といった暗号化が可能です。
　パスフレーズによるシンプルな暗号化にも対応しています。


■動作環境

　Windows 10
　確認していませんが Windows 7/8/8.1 でも動くかもしれません。

　.NET Framework 4 を使用します。


■インストール方法

　アーカイブの中身をローカルディスク上の任意の場所にコピーして下さい。


■アンインストール方法

　レジストリなどは一切使っていません。
　ファイルを削除するだけでアンインストールできます。


■使い方

　●圧縮

　　以下の何れかの方法で圧縮できます。

　　1. 圧縮したいフォルダ・ファイルを "圧縮.exe" にドロップする。

　　2. デスクトップなどに "圧縮.exe" のショートカットを作成し、
　　   圧縮したいフォルダ・ファイルを作成したショートカットにドロップする。

　　3. SendTo に "圧縮.exe" のショートカットを作成し、
　　   圧縮したいフォルダ・ファイルを右クリック / 送る / 圧縮.exe のショートカットを選択する。

　　4. "圧縮.exe" を（引数無しで）実行し、
　　   圧縮したいフォルダ・ファイルをウィンドウにドロップする。★要設定変更

　　5. "圧縮.exe" を（引数無しで）実行してウィンドウをクリックし、
　　   圧縮したいフォルダ・ファイルを「フォルダ・ファイル選択ダイアログ」から選択する。★要設定変更


　　デフォルトの設定の場合、フォルダ・ファイルと同じ場所にアーカイブファイルを作成します。


　●展開

　　以下の何れかの方法で展開できます。

　　1. 展開したいアーカイブファイルを "展開.exe" にドロップする。

　　2. デスクトップなどに "展開.exe" のショートカットを作成し、
　　   展開したいアーカイブファイルを作成したショートカットにドロップする。

　　3. SendTo に "展開.exe" のショートカットを作成し、
　　   展開したいアーカイブファイルを右クリック / 送る / 展開.exe のショートカットを選択する。

　　4. "展開.exe" を（引数無しで）実行し、
　　   展開したいアーカイブファイルをウィンドウにドロップする。★要設定変更

　　5. "展開.exe" を（引数無しで）実行してウィンドウをクリックし、
　　   展開したいアーカイブファイルを「フォルダ・ファイル選択ダイアログ」から選択する。★要設定変更


　　デフォルトの設定の場合、アーカイブファイルと同じ場所にファイル・フォルダを展開します。


　●設定

　　"圧縮.exe", "展開.exe" 又は WCluster.exe を（引数無しで）実行して下さい。
